## 2024-11-23 - FastAPI Async Optimizations\n**Learning:** Synchronous network calls (like Groq) and database ops in `async def` FastAPI endpoints block the entire event loop, causing poor concurrency. SQLite `connect(..., check_same_thread=False)` is used here, making `asyncio.to_thread` safe for DB calls.\n**Action:** Use AsyncGroq for network I/O, and `asyncio.to_thread` for SQLite DB I/O in async endpoints.
